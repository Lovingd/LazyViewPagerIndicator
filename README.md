# LazyViewPagerIndicator

ViewPager+底部导航栏（懒加载）
添加在你的根build.gradle在库底：

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
  添加依赖关系
  
  dependencies {
	        compile 'com.github.Lovingd:LazyViewPagerIndicator:1.0.1'
	}
